# 🔐 Password Strength Checker

A web-based application that evaluates password strength and provides suggestions for improvement.

## 🚀 Features
- Checks password length and complexity
- Detects weak passwords
- Suggests strong password alternatives
- User signup system with password hashing

## 🛠 Tech Stack
- Frontend: HTML, CSS
- Backend: Python (Flask)
- Database: SQLite

## 🔒 Security Features
- Password hashing using SHA-256
- Input validation
- Basic authentication system

## 📸 How it Works
1. Enter a password
2. System evaluates strength
3. Displays suggestions
4. Generates strong password if weak
